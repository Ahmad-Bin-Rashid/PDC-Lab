# Parallel and Distributed Computing Labs
A comprehensive collection of lab tasks, implementations, experiments, and reports for the Parallel and Distributed Computing course.

---

## How to Use This Repository

1. Navigate to a specific lab folder.
2. Refer to the lab-specific `README.md` for instructions.
3. Compile or run the programs in the `src/` directory.
4. Review execution results and analysis in the `report/` directory.

---

## Purpose of This Repository

- Maintain organized lab work across multiple experiments
- Serve as a reference for parallel programming techniques
- Act as an academic and technical portfolio
